SQL 명령어

  • sqlite3 (이름).db : 데이터베이스 생성
  • CREATE TABLE {이름}( id integer, name text, ) : table 생성
  • .table : 생성된 table 조회
  • SELECT * FROM {테이블 이름};: 테이블 내용 조회
    • WHERE 절
    • GROUP BY 절, 앞에 조건을 걸때는 WHERE, 뒤에 조건을 걸때는 HAVING
    • LIMIT 절
  • INSERT INTO {테이블 이름} VALUES (데이터값); : 데이터 삽입
  • UPDATE {테이블 이름} SET (신규데이터값) WHERE (기존데이터값); : 데이터 삽입
  • DELETE FROM {테이블 이름} WHERE (데이터값); : 데이터 삭제
  • DROP TABLE {테이블 이름}; : 테이블 삭제
